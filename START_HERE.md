## 💬 質問・サポート

### よくある質問

**Q: TypeScript が初めてですが大丈夫ですか？**  
A: ドキュメント内のコードはすべてコピペ可能です。型定義も完全版が提供されています。

**Q: Prisma って何ですか？**  
A: PostgreSQL を簡単に使えるようにする ORM（データベースツール）です。SQL を書かなくても TypeScript でデータベース操作ができます。

**Q: Docker がインストールされていません**  
A: クイックスタートガイドの「トラブルシューティング」セクションを参照してください。

**Q: 既存のコードを壊してしまいそうで心配です**  
A: Git でブランチを作成してから作業を始めることをお勧めします：
```bash
git checkout -b feature/schema-types
```

**Q: バックエンドを先に作るべきですか？フロントエンドを先に作るべきですか？**  
A: 並行開発が可能です。「今すぐ始めるアクションリスト」でフロントエンドを拡張しながら、別のブランチで「クイックスタートガイド」に従ってバックエンドを構築できます。
