## 🎯 プロジェクトの目標

**無人航空機日誌システム** - 国土交通省の「無人航空機の飛行日誌の取扱要領」に完全準拠した飛行記録管理システムを構築する。

### 必須機能
1. ✅ **様式1**: 飛行記録
2. ✅ **様式2**: 日常点検記録
3. ✅ **様式3**: 点検整備記録
4. ✅ **データ入出力**: CSV / Excel / PDF
5. ✅ **3年間保存**: 法的義務への対応

## 🚀 次のアクション

### 即座に開始可能: Phase 1

**Phase 1: データベース設計・構築（推定3日）**

すでに準備完了：
- ✅ `docker-compose.yml` 作成済み
- ✅ Prisma スキーマ設計済み（`クイックスタートガイド.md` 参照）
- ✅ `.gitignore` 作成済み

必要な作業：
```bash
# 1. PostgreSQL を起動
docker-compose up -d

# 2. バックエンドディレクトリを作成
mkdir backend
cd backend
npm init -y

# 3. 依存関係をインストール（詳細は クイックスタートガイド.md を参照）
npm install prisma @prisma/client express ...

# 4. Prisma をセットアップ
npx prisma init

# 5. schema.prisma を編集（クイックスタートガイドからコピー）

# 6. マイグレーション実行
npx prisma migrate dev --name init

# 7. Prisma Studio で確認
npx prisma studio
```
