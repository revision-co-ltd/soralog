**旧アーキテクチャ** (アーカイブ済み):
```
Frontend → Express Backend → Prisma ORM → PostgreSQL (Docker)
```

**現在のアーキテクチャ**:
```
Frontend → Supabase (PostgreSQL + Auth + REST API)
```

削除せずに保存する理由：

4. **コンテキスト** - なぜ Supabase に移行したかの背景
