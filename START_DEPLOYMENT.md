## ⚡ 超级快速开始（3步）

### 第1步：准备数据库（2分钟）

访问 https://supabase.com
```bash
1. 注册/登录
2. 创建新项目
3. 复制数据库URL
```

### 第2步：推送到GitHub（2分钟）

```bash
cd "/Users/yang/Downloads/20251113无人机飞行记录APP "
git init
git add .
git commit -m "初始提交"
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
```

### 第3步：部署到Vercel（2分钟）

访问 https://vercel.com
```bash
1. 导入GitHub仓库
2. 添加环境变量：
   - DATABASE_URL=你的数据库URL
   - JWT_SECRET=随便一个长字符串
   - VITE_API_BASE_URL=https://your-backend.railway.app/api
3. 点击Deploy
```

### 第4步（可选）：部署后端到Railway

如果你需要后端API：

访问 https://railway.app
```bash
1. 导入同一个GitHub仓库
2. 添加环境变量（同上）
3. 设置启动命令：npm run backend
4. 部署
```

## 🆘 遇到问题？

### 问题：数据库连接失败
**解决：** 检查DATABASE_URL是否正确，确保包含 `?sslmode=require`

### 问题：页面空白
**解决：** 
1. 检查浏览器控制台是否有错误
2. 确认API地址配置正确
3. 运行数据库迁移：`npx prisma migrate deploy`

### 问题：API请求失败
**解决：**
1. 确认后端已部署
2. 检查CORS配置
3. 确认环境变量正确

### 更多问题？
查看 [VERCEL_部署指南.md - 常见问题](./VERCEL_部署指南.md#-常见问题排查)
