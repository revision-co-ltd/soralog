# 🎨 ソラログ (SoraLog) ブランディングガイド

## ブランド概要

### アプリ名
- **日本語**: ソラログ
- **英語**: SoraLog
- **読み**: そらろぐ
- **意味**: 
  - **ソラ (空/Sora)**: 天空、空中、ドローンが飛ぶ空間
  - **ログ (Log)**: 記録、日誌

### コンセプト
```
空を飛ぶ記録 = ソラログ
シンプルで覚えやすく、日本語と英語の両方で通用
```

## よくある質問（FAQ）

### Q1: 「ソラログ」という名前の由来は？
```
A: 「ソラ（空）」と「ログ（記録）」を組み合わせた造語です。
   ドローンが飛ぶ空間と、その飛行記録を管理するシステムを
   シンプルに表現しています。
```

### Q2: 無料で使えますか？
```
A: 基本機能は無料でご利用いただけます。
   （将来的にエンタープライズプランも検討中）
```

### Q3: どんなデバイスで使えますか？
```
A: スマートフォン、タブレット、PCのいずれでも使用可能です。
   特にスマートフォンでの使いやすさに配慮して設計しています。
```
