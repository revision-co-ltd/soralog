## ✨ 特别提示

### ⚠️ 环境变量
永远不要提交 `.env` 文件到Git！
已配置 `.gitignore` 自动忽略。

### ⚠️ 数据库迁移
部署后记得运行：
```bash
npx prisma migrate deploy
```

### ⚠️ CORS配置
确保后端配置了正确的CORS：
```env
CORS_ORIGIN=https://your-app.vercel.app
```
